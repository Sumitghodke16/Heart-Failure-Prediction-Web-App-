## 📌 Overview
This project is an end-to-end Machine Learning application that predicts the risk of heart failure based on clinical features.

It includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training using Logistic Regression
- Deployment as an interactive web application using Streamlit

---

## 🚀 Features
- 📊 Data Analysis and Visualization (EDA)
- 🤖 Machine Learning Model (Logistic Regression)
- 🌐 Interactive Web Application (Streamlit)
- ⚡ Real-time prediction based on user input
- 🧠 Uses medical parameters like age, serum creatinine, ejection fraction, etc.

---

## 🧠 Machine Learning Workflow

### 1. Data Preprocessing
- Handled missing values
- Feature selection
- Data cleaning

### 2. Exploratory Data Analysis (EDA)
- Distribution analysis
- Correlation analysis
- Feature importance understanding

### 3. Model Building
- Algorithm used: **Logistic Regression**
- Train-test split applied
- Model trained and evaluated using accuracy

### 4. Model Deployment
- Built using **Streamlit**
- User-friendly interface for prediction
- Takes real-time input and returns prediction
